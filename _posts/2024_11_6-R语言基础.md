## while循环,repeat循环和for循环的辨析

![image](https://github.com/user-attachments/assets/f6a837a1-3d64-49ae-9ace-abceac1c67e5)

为了方便检查，我们设置随机种子，让样本取的一样。最后发现三种方法的结果一致：
![image](https://github.com/user-attachments/assets/958de7d8-90f5-4823-8388-e4be420a7f7e)
# repeat循环
```r
set.seed(0)
x <- seq(10,1000,1)
u <- c()#u的长度不知道，是一个空向量
i <- 1
repeat{
  u[i] <- t.test(rnorm(x[i],0.001,1),m=0,alt="greater")$p.value
  if(u[i]<0.05)break
  i <- i+1
}
输出结果
i
x[i]
u
```
# while循环
```r
set.seed(0)
x <- seq(10,1000,1)
u <- c()#u的长度不知道,是一个空向量
i <- 1
u[i] <- t.test(rnorm(x[i],0.001,1),m=0,alt="greater")$p.value
#要先有值才能比较
while(u[i]>=0.05){
  i <- i+1
  u[i] <- t.test(rnorm(x[i],0.001,1),m=0,alt="greater")$p.value
 
}
#输出结果
i
x[i]
u
```

# for循环
```r
set.seed(0)
x <- seq(10, 1000, 1)  # x 是样本大小的序列
u <- numeric(length(x))  # 预先初始化 u 为一个零向量
# 使用 for 循环，查找第一个 p 值小于 0.05 的位置
for (i in  1:length(x)) {
  # 生成正态分布样本，并进行 t 检验
  u[i] <- t.test(rnorm(x[i], mean = 0.001, sd = 1), mu = 0, alt = "greater")$p.value
  
  # 如果 p 值小于 0.05，跳出循环
  if (u[i] < 0.05) {
    break
  }
}
#输出结果
i
x[i]
u[1:i]
```
